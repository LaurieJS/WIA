What is Spritely?

  • Spritely is a jQuery plugin created by ArtLogic
  • It's easy to use and lightweight
  • Works on all major browsers (even IE 6)
  • Also works on iPhones, iPods, and iPads
  • Great alternative to Flash
  • It's free! "Dual licensed under the MIT GPL Version 2 licenses"

What does Spritely do?

  • Uses HTML and jQuery to create animations
  • Helps to create dynamic scrolling background animations
  • Makes animating sprites quick and easy
  • Makes animations interactive and draggable
  • Find out more at the Spritely.net website
  • Websites using Spritely: example 1 / example 2 / example 3

Setting up the Files

The basic HTML file setup for Spritely:

<!DOCTYPE html>
<html>
<head>
  <title>Spritely Tutorial by Laurie</title>
  <!--JavaScript Files needed for Spritely-->
  <script type="text/javascript" src="js/jquery-1.11.1.min.js"></script>
  <script type="text/javascript" src="js/jquery-ui.min.js"></script>
  <script type="text/javascript" src="plugins/spritely/src/jquery.spritely.js"></script>
  <!--Custom CSS file-->
  <link href="css/styles.css" rel="stylesheet"/>
</head>

<body>
  <!--Custom JavaScript file-->
  <script src="js/main.js" type="text/javascript"></script>
</body>
</html>

Spritely HTML Code

Add the following code inside the body tag of the HTML file:

<body>
  <!--Add the following code inside the body tag-->
  <div id="stage" class="stage">
    <div id="background" class="stage"></div>
    <div id="unicorn" class="stage"></div>
    <div id="planets" class="stage"></div>
  </div><!--end of the stage div-->
</body>
  • A div with an id and class of "stage" wraps around the div elements
  • A div with a class of "stage" for all of the elements that will be on the stage
  • The div elements are given an id so they are easy to target with JQuery

Background Image of the Stars

  • The background image can be any size
  • Not all images need to be animated
  • Save as a transparent .png file
  • Back to images

Image of the Planets

  • Can be any size but should be smaller than the background
  • Leave space on the end(s) of the artboard so the first and last objects will not be side-by-side when animated
  • Make sure to choose the "clip to artboard" option while saving, or the first and last object will be side-by-side
  • Save as a transparent .png file
  • Back to images

Unicorn Sprites

The sprites that I created each fit in a space 180px by 150px. The total dimensions of the image are 540px by 150px

  • Make all of the sprites the same size
  • Make sure that the sprites are aligned properly
  • Bring the sprites to life by giving them each a different pose
  • Save as a .png file with a transparent background
  • That's all there is to it! Back to images

jQuery Code

$(document).ready(function() {
 $("#unicorn").sprite({ fps: 3, no_of_frames: 3 }).isDraggable();
 $("#planets").pan({fps: 25, speed: 2, dir: 'left'});
 $("#background").pan({fps: 5, speed: 2, dir: 'left'});
});
  • Sprite Code Info
  • The sprite method animates the sprites at a speed desired
  • fps: represents the number of frames shown per second
  • no_of_frames: indicates the number of sprites being used
  • isDraggable: a JQuery UI method that makes elements draggable
  • Pan Code Info
  • The pan method moves images across the screen in the direction and speed desired
  • speed: is how fast the image will move
  • dir: is the direction the image will move
